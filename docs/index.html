Quick Start

Installation

  1. Download the latest installer
  2. Run PlaylistSetup.exe
  3. Launch Playlist from your Start Menu

Creating Your First Playlist

  1. Click File ‚Üí New Playlist
  2. Give it a name
  3. Click Add Files/Folders
  4. Select your media files
  5. Click Save

Playing Media

  • Double-click any item to start playing
  • Right-click ‚Üí Continue to resume from where you stopped
  • Use F11 for full-screen mode
  • Press Space to pause/play

System Requirements

  • Windows 10 or Windows 11 (64-bit)
  • .NET 9.0 Runtime (included in installer)
  • 2 GB RAM minimum
  • 100 MB disk space for application
  • VLC Libraries (included in installer)
